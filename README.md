# inosoft-vechicle
Inosoft Test Vehicle REST API
How to run
1. open terminal
2. initiate composer 
```bash
composer install
```
3. setting your .env with prepared clean database
4. now run 
```bash
php artisan migrate
```
5. go to public directory
```bash
cd /public
```
6. serve app with command :
```bash
php -S localhost:8000
```
7. test with postman or others API tester.
